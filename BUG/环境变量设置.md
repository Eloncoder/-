## 查看环境变量
```
echo $LD_LIBRARY_PATH
echo $LIBRARY_PATH
```

## 设置环境变量
```
LD_LIBRARY_PATH=NEWDIRS:$LD_LIBRARY_PATH   （newdirs是新的路径串）
LIBRARY_PATH=NEWDIRS:$LIBRARY_PATH         （newdirs是新的路径串）
```
